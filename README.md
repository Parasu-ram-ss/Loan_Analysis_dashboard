# Loan_Analysis_dashboard

# Project Objective

The Loan Analysis Dashboard is developed using Microsoft Power BI to analyze loan data and monitor lending performance. The dashboard provides insights into loan approvals, repayments, defaults, and overall financial performance.


# Dataset Used

The dataset includes applicant details, loan amount, loan status, interest rate, loan term, repayment details, and credit score information.

•	Microsoft Power BI Desktop (.pbix)

•	Power Query for Data Transformation

•	Data Modeling and Relationships

•	DAX (Data Analysis Expressions)
